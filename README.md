# GetcarSystem

Sistema direcionando para o controle e gerência de veículos de uma locadora de mesmo segmento.

# Requisitos técnicos do projeto

- Ruby 2.4.*
- Rails 5.1.2
- PostgreSQL >= 9.5.7
